# nci-tools
Admin tools for managing ARCCSS accounts at NCI

## gdata_report

Print a listing of storage use on /g/data, including the size of data that
hasn't been used for > 3 months

```
$ gdata_report
Project                            Used (GB)     Old (GB) 
-----------------------------------------------------------
ACCESS-OM                                 2            2   (100%)
-----------------------------------------------------------
Total                                     2            2   (100%)
```
